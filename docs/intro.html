Get started¶

Installation¶

In order to install DHTK, Python 3.x is required. Note that when using a local endpoint, Docker application is required as DHTK uses a Fuseki container as endpoint.

There are two ways to get DHTK python module:

1) Download the DHTK source code in your preferred format, extract the DHTK files and access them via console.

$ cd dhtk
$ python setup.py

2) Use package installer for Python (pip):

$ pip install git+https://gitlab.com/dhtk/dhtk.git

How to use¶

DHTK is a module for Python3.x. As such, it is accessible using any Python console or environment. DHTK access occurs in 4 steps:

1) Import DHTK module

import dhtk
# ###################################################
# Welcome to the Digital Humanities ToolKit,
# the  user - friendly Python API for
# Digital Humanities research.
#
# For more information, please visit DHTK 's website:
# [https://dhtk.unil.ch/]
# ####################################################

2) Define DHTK settings

configs = dhtk.start(dataset="gutenberg")
# Current DHTK settings:
# [user]: s05mm2
# [wd]: /Users/s05mm2/DHTK/s05mm2
# [endpoint]: localhost
# [dataset]: gutenberg
# [modules]: {'gutenberg': datetime.datetime(2020, 12, 16, 13, 52, 26, 965388)}
# [_file]: /Users/s05mm2/DHTK/s05mm2/s05mm2.config
# [logger]: <Logger dhtk(DEBUG)>
# [_connect]: 30

Settings can be defined by passing then as argument at instantiation (dhtk.start(**kwarg)) or they can be modified using the dot notation. .. literalinclude:: ./examples/usage.py

The main settings are user and wd, respectively the user name and the DHTK’s working directory. These two settings are used to search for saved configurations and correctly organize the logs and save the search results in the DHTK folder structure.

3 a) Connect to a `SPARQL <https://en.wikipedia.org/wiki/SPARQL>`_ endpoint

If you’re using a remote SPARQL endpoint, you can provide it as a URL (website) using the endpoint setting. After DHTK is set up, this step is simply connects to the SPARQL endpoint to use. As such, it is integrated with the next step for moist user cases. However, note that this step performs important checks to ensure DHTK is properly set up.

3 b) Load the required DHTK extension

# [_connect]: 30
gutenberg = dhtk.get_module(configs)
# DHTK Gutenberg statistics:
# Last update: 16 Dec 20
# number of books: 56913
# number of authors: 21031
# number of bookshelves: 337
# number of subjects: 34045

Note that the dataset setting, defining the DHTK extension module to use, must be defined for the first time, otherwise DHTK is not aware of which module to load!

Once the module is loaded, DHTK functionalities are ready to use!

4) Query the dataset using `DHTK extended functionalities <./architecture.html#extension-modules>`_. Current extensions include:

  • Project Gutenberg

  • Auchinkleck manuscript

Gutenberg¶

DHTK uses a blueprint for the development of all extension modules, simplifying and unifying the different functionalities. Using this framework, the Gutenberg extension dataset allows you to search Gutenberg.org for books, authors, bookshelves and subjects, retrieving all the available information on these as book Corpus that can be saved.

1. Searching for an author¶

gutenberg.get(what="author", name="Lovecraft, Howard Phillips", add=True)
# ['Lovecraft, H. P. (Howard Phillips) (Lovecraft, Howard Phillips)']

2. Searching for an subject¶

gutenberg.get(what="subject", name="Horror films", add=True)
# ['[SUBJECTS: Horror films]: Night of the Living Dead  by Romero, George A.']
gutenberg.get(what="shelves", name="FR Chansons", add=True)

3. Searching for an bookshelf¶

gutenberg.get(what="shelves", name="FR Chansons", add=True)
# ['[SHELF: FR Chansons]: La chanson des joujoux  by Jouy, Jules',
# '[SHELF: FR Chansons]: Recueil de chansons en patois de la Bresse  by La Gelière, P.-C. de',
# '[SHELF: FR Chansons]: Le chanteur parisien\nRecueil des chansons de L.A. Pitou  by Pitou, Louis Ange']

4. Searching for an book¶

gutenberg.get(what="books", name="The King in Yellow", add=True)
# ['The King in Yellow by Chambers, Robert W. (Robert William) (en)']

As you can see, a (Python) dictionary-like syntax can be used to search the dataset using only a small number of arguments:

  • what to search for.

  • name of what is being searched. If no name is provided, all available options are retrieved.

  • add to corpus or just check results first?

8. Save Corpus¶

Before saving the Corpus to disk (books and Python object), it is important to change the corpus descriptions, particularly the corpus name, which is used as the directory where the books will be stored.

# Modify Corpus descriptions
gutenberg.corpus(name="Random horror",
                 description="Horror stuff from Gutenberg (french songs included!)")
# Corpus name:
# 	Random horror
# Corpus description:
# 	Horror stuff from Gutenberg (french songs included!)
# Corpus path:
# 	/Users/s05mm2/DHTK/s05mm2/requests/Random_horror
# Corpus books:
# 0	Howard Phillips Lovecraft    	The Dunwich Horror
# 1	Howard Phillips Lovecraft    	The Shunned House
# 2	Howard Phillips Lovecraft    	Writings in the United Amateur, 1915-1922
# 3	George A. Romero             	Night of the Living Dead
# 4	Jules Jouy                   	La chanson des joujoux
# 5	P.-C. de La Gelière          	Recueil de chansons en patois de la Bresse
# 6	Louis Ange Pitou             	Le chanteur parisien Recueil des chansons de L.A. Pitou

# Save Corpus to disk
gutenberg.save()

Search metadata¶


from dhtk.catalogs.gutenberg.data import GutenbergData
from dhtk.metadata.dbpedia import DbpediaMetadata

# Initialise class GutenbergData as gutenberg_search
gutenberg_search = GutenbergData()

# search the book using gutenberg_id"""
book = gutenberg_search.get_book('http://www.gutenberg.org/ebooks/2489')

# Initialise class DbpediaMetadata as dbpedia_metadata
dbpedia_metadata = DbpediaMetadata()

# get the metadata of book from bdpedia and add it to the book_id
dbpedia_uri = dbpedia_metadata.get_books_metadata_uri(book)
metadata = dbpedia_metadata.get_book_metadata(book)
book.update_metadata(metadata)

# print the medata of book from dbpedia
pprint(book.metadata)
