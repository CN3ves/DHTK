AbstractExtension()¶

Abstract class to be used as template for new extension modules __init__ script

class dhtk.extensions.blueprint.AbstractExtension(settings, client)¶

Abstract Class that serves as a blueprint for the DHTK extension modules and additional datasets.

name: string

Name of extension

url: string

URL to the extension RDF file

query: A function wrapper unifying the extension utilities

set_up() → None¶

Method to retrieve the DHTK extension dataset stored on Zenodo and upload it to the local Fuseki server

update() → None¶

Method to check for remote changes to the dataset RDF file and update the local Fuseki endpoint

This blueprint can be used to wrap the new modules into DHTK. For more information, see Community
