Settings()¶

DHTK setting and configurations module

class dhtk.core.settings.Settings(user: str = 's05mm2', wd: Union[str, pathlib.PosixPath] = '~/DHTK', endpoint: str = None, dataset: str = None, modules: dict = <factory>)¶

Data class to manage DHTK’s settings and behaviour.

wd: string or pathlib.PosixPath [default: pathlib.Path.home() / “DHTK”]

DHTK working directory to save logs, setting, configurations and temporary files

user: string [default: pathlib.Path.home().name]

Name for user to distinguish local users in shared machines

endpoint: string [default: “localhost” or from saved configurations]

Either the URL for the remote triplestore client or local Docker/Fuseki triplestore

dataset: string [default: “” or from saved configurations]

User’s default dataset from available DHTK extensions to be queried NOTE that dataset must be set at least once!!

Module for setting up and connecting to triplestore

class dhtk.core.client.Client(settings, install: bool = False)¶

Class to manage DHTK connection to the triplestore endpoint.

Instantiates an object that tries to connect to the required triplestore. If the triplestore is local, it sets up a Fuseki container on Docker.
